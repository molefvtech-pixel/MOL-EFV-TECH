# MOL EFV TECH

Sitio web estático listo para GitHub Pages.

## Estructura

- `index.html`
- `css/style.css`
- `css/responsive.css`
- `js/script.js`
- `img/logo.png`
- `img/gamer.png`
- `img/sistemas.png`
- `img/mascota-koala.mp4`
- `audio/musica.mp3`

## Probar en VS Code

1. Instala la extensión **Live Server**.
2. Abre `index.html`.
3. Clic derecho → **Open with Live Server**.

## Publicar en GitHub Pages

1. Crea un repositorio público en GitHub.
2. Sube todo el contenido de esta carpeta manteniendo la estructura.
3. En el repositorio entra a **Settings → Pages**.
4. En **Build and deployment**, selecciona **Deploy from a branch**.
5. Selecciona la rama `main` y la carpeta `/ (root)`.
6. Guarda y espera a que GitHub publique la URL.

## Música

El sitio intenta reproducir la música a volumen bajo (12%). Los navegadores pueden bloquear el audio automático con sonido. Si eso ocurre, el sitio muestra un botón para que el visitante la active.
